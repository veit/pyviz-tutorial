GeoPandas
=========

`GeoPandas <https://geopandas.org/en/stable/>`_ extends :doc:`../pandas/index`
with data types that enable spatial operations on geometric types, whereby the
geometric operations are carried out by `Shapely
<https://shapely.readthedocs.io/en/stable/>`_. GeoPandas uses `Fiona
<https://fiona.readthedocs.io/en/stable/>`_ for data access and `descartes
<https://pypi.org/project/descartes/>`_ and :doc:`../index` for plotting.
