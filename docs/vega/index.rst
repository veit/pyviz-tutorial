Vega
====

`Vega <https://vega.github.io/vega/>`_  is a declarative language for creating,
saving and sharing interactive visualisation designs. With Vega, you can
describe the visual appearance and interactive behaviour of a visualisation in a
JSON format and generate web-based views with Canvas or SVG.
