PdVega
======

`PdVega <https://altair-viz.github.io/pdvega/>`_ is a library that can be used
to create interactive `Vega-Lite <https://vega.github.io/vega-lite/>`_ plots
from pandas data frames using the pandas `plot
<https://pandas.pydata.org/pandas-docs/stable/user_guide/visualization.html>`_
API.

.. seealso::

    * `Advanced Plotting: Using Vega-Lite Directly
      <https://altair-viz.github.io/pdvega/advanced.html>`_
    * `API Reference <https://altair-viz.github.io/pdvega/API.html>`_
